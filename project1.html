Project 1 - Currency Exchange

Add $$$

      I decided to implement some plain JavaScript with an API of a currency exchange. Data is returned with the base of EURO, so that is why the first box (base currency) is EURO by default. You can play around with adding more currencies from the list by clicking on the "ADD $$$" button. You can also remove them from the list.
